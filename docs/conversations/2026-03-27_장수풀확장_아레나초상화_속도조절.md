# 장수 풀 확장 + 아레나 초상화 + 속도 조절

## 날짜: 2026-03-27

## 주요 작업 내역

### 1. 장수 풀 50명으로 확장
- 기존 22명 → 50명
- UR 6명 (유지), SSR 12명 (+4), SR 15명 (+7), R 17명 (신규)
- 신규 SSR 고유 스킬 5종: 여몽(백의도강), 장합(교묘한전술), 위연(자오곡기습), 태사자(일기토)
- R등급: 하후무/조진/우금/왕랑/이엄/미축/간옹/여범/주태/반장/한당/장임/고람/심배/정욱/장수/공손찬

### 2. 일반 가챠 확률 변경
- 변경 전: UR 1%, SSR 9%, SR 90%
- 변경 후: R 60%, SR 30%, SSR 10% (UR 불가)
- UR은 프리미엄 전용으로 희소성 유지

### 3. 가챠 장수 목록/확률 표시
- 가챠 화면에 "📋 확률/장수 목록 보기" 버튼
- 등급별(UR/SSR/SR/R) 전체 장수 목록 표시
- 각 장수: 등급, 이름, 병종, 고유 스킬

### 4. 아레나 전투 초상화 스타일
- 원형 프레임 + 등급 색상 테두리
- 이미지 있는 장수: 원형 마스크로 크롭
- 이미지 없는 장수: 병종 아이콘 표시
- 진영 표시 (파란점=아군, 빨간점=적군)
- HP 바 + MP 바 표시

### 5. 아레나 전투 속도 조절
- 기본 속도 2배 느리게 (100~400ms → 400~800ms)
- 배속 버튼 (1x/2x/3x) 전투 화면 우상단 추가

### 6. 아레나 스킬 데미지 수정
- 물리 스킬 (돌격 등): 공격력 기반
- 마법 스킬 (화계 등): 정신력 기반
