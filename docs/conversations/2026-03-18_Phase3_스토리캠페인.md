# 2026-03-18 | Phase 3 - 스토리 캠페인 + 버그 수정

## 구현 완료
- 대화 시스템 (DialogueScene): 화자명 + 대사, 클릭 진행
- 캠페인 데이터: 1장 황건적의 난 (3스테이지, 대화/전투/보상)
- WorldMapScene: 챕터/스테이지 선택, 상태 표시 (완료/가능/잠금)
- TitleScene: 새 게임/이어하기
- CampaignManager: localStorage 세이브/로드, 유닛 영속, 전투 간 HP 회복
- 씬 흐름: Title → WorldMap → Dialogue → Battle → Dialogue → WorldMap

## 버그 수정
1. 스킬/공격 버튼 클릭 안 됨 → _menuClickConsumed 플래그로 이벤트 충돌 해결
2. 씬 전환 후 클릭 안 됨 → scene.stop() + scene.start() 패턴을 scene.start()만으로 변경
3. 챕터 완료 후 빈 화면 → 챕터 없을 때 "완료" 메시지 표시
4. BattleScene 재시작 시 클릭 안 됨 → create()에서 모든 상태 변수 리셋
5. UnitInfoPanel이 클릭 가로챔 → pointerEvents: 'none' 추가
6. 제자리 클릭/이동 취소/적 범위 미리보기 기능 추가
