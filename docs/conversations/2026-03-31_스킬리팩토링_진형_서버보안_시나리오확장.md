# 2026-03-31~04-03 스킬 리팩토링, 진형 시스템, 서버 보안, 시나리오 확장

## 주요 작업

### 1. 스킬 시스템 리팩토링
- 복합 효과 스킬 구조 (SkillSubEffect[]) 추가 — 하나의 스킬에 데미지+버프+디버프 조합 가능
- 상태효과 타입 정의 (StatusEffectId) — 19종 (버프 8 + 디버프 11)
- 데미지 스케일링: attack 기반(물리) vs spirit 기반(술법)
- 기존 40+ 스킬 레거시 호환 유지 + 19개 고유 스킬 복합 효과 업그레이드
- 새 장착 스킬 9종 추가 (물리/술법/지원)
- CombatSystem: speed/morale/move/range 상태효과 반영

### 2. 스킬 강화 시스템 (Lv.1~10)
- 3단계 스킬서: 초급(Lv.1-3), 중급(Lv.4-6), 고급(Lv.7-10)
- 레벨당 +10% 위력, -1 MP, Lv.5/10에서 쿨다운 -1
- 병종 스킬은 강화 불가 (승급으로만)
- 서버 검증 (POST /save/enhance-skill)
- 상점에 스킬서 판매 추가

### 3. 모드별 스킬 슬롯 제한
- 시나리오: 병종 + 고유 (2개만)
- PvP/던전: 전체 4슬롯

### 4. 진형(진법) 시스템
- 3×3 패턴 기반 배치 시스템 (5가지 진형)
- PvP 배치 + 일일 던전에 적용 (시나리오 제외)
- 진형 패턴에 맞게 장수 배치 → 버프 활성화 (3턴)
- 일일 던전: 자동 선택 → 3×3 배치 시스템으로 변경

### 5. 서버 보안 마이그레이션 (Phase 1~5)
- Phase 1: 금화 서버 컬럼 + 세이브 보안 강화 (gold strip)
- Phase 2: 상점 구매 서버 검증 (POST /save/shop-buy)
- Phase 3: 승급/각성 서버 검증 (POST /save/promote, /save/awaken)
- Phase 4: 던전/전투 보상 서버 검증
- Phase 5: 일일 임무/출석 보너스 서버 검증
- 총 11개 서버 엔드포인트 추가

### 6. 시나리오 챕터 3~10 완성
- 8개 챕터, 21개 스테이지 추가 (총 29 스테이지)
- 삼국지 원작 네임드 장수 적군으로 추가
- 난이도 상향 (Ch3 Lv8~12 → Ch10 Lv28~35)
- 최종 전투: 관우 Lv35 HP550/ATK100 등 8명 보스

### 7. PvP 아레나 전투 연출 강화
- 전장 배경 (하늘+대지 그라데이션)
- VS 컷인, 턴 팝업, 행동 순서 바
- 공격 돌진 + 화면 흔들림 + 임팩트 플래시
- 스킬 컷인 (어두운 오버레이 + 금색 텍스트)
- 결투 클로즈업 (공격/피격/사망 PixelLab 애니메이션)
- 원거리 유닛 제자리 공격
- 사망 회전+페이드+격파 텍스트
- 승리/패배 파티클 이펙트

### 8. 일일 던전 전투 연출 강화
- 전장 배경 + 보스 웨이브 붉은 오버레이
- PixelLab 스프라이트 유닛 카드
- 공격 40px 돌진 + 임팩트 플래시
- 웨이브 전환 컷인
- 결과 화면 바운스 + 별 순차 등장

### 9. 기타
- 유닛 정보 패널: 병종별 핵심 스탯 + 펼치기 기능
- 장수 상세: 전체 12개 스탯 표시
- 전투 포기 버튼 추가
- 행동 완료 유닛 클릭 시 정보 표시
- PvP 상태 서버 저장 (ELO/전적/일일 티켓)
- 가챠 뽑기 후 장수 안 보이는 버그 수정

## 기술적 결정
- 스킬 복합 효과: effects[] 배열 + 레거시 호환 (effectType/power 유지)
- 진형: 시나리오 제외, PvP/던전만 적용 (3×3 패턴 기반)
- 서버 보안: 점진적 마이그레이션, 금화→상점→승급/각성→던전→임무 순서
- 스킬 강화: 병종 스킬 불가, 고유/장착만 가능
- PvP 클로즈업: 스킬 사용 시에만 발동, 원거리 유닛 제자리
