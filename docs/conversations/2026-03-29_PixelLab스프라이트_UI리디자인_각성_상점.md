## 기술적 결정
- PixelLab 캐릭터는 4방향(south/east/north/west)으로 통일 (크레딧 절약)
- 커스텀 애니메이션: 방향당 ~20 gen, 4방향 = ~80 gen/종
- 기병 캐릭터: quadruped horse 대신 humanoid chibi로 결정 (치비 일관성)
- 승급: 전투 중 자동 → 인수 아이템 소비 수동으로 변경
- 가챠 중복: 금화 변환 → 장수 조각 저장으로 변경 (각성 시스템 연동)
