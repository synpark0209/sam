# PvP 아레나 구현 + 스탯 시스템 확장 + UI 개선

## 날짜: 2026-03-26

## 주요 작업 내역

### 1. PvP 아레나 구현
- PvPArenaScene 생성 (로비 → ⚔️ PvP 아레나)
- 3x3 그리드 배치 (후열/중열/전열, 가로 배치)
- 장수 선택 → 원하는 칸 클릭으로 배치
- 자동 전투 엔진 (속도순 행동)
- 전열 보호 시스템 (앞열 전멸 시 뒤열 공격 가능, 궁병/책사는 직접 공격)
- 상성 적용 (1.3x/0.7x)
- 시각 이펙트 (공격 돌진, 데미지 팝업, 피격 흔들림, 상성 라벨)
- 실시간 전투 로그 (하단 5줄)
- 병종 아이콘 (🐎🛡️🏹📜👊🗡️)

### 2. PvP 랭킹/시즌 시스템
- ELO 기반 랭킹 (K=32)
- 5단계 티어: 브론즈(~999) → 실버(1000+) → 골드(1200+) → 다이아(1400+) → 마스터(1600+)
- 아레나 홈 화면 (티어/ELO/전적/진행도 바)
- 일일 PvP 티켓 5개
- 시즌 보상 테이블 (금화/보석/스킬/장비, 마스터는 적토마)

### 3. 스탯 시스템 대폭 확장 (6→12 스탯)
- 신규 6개: spirit(정신력), agility(민첩), critical(순발력), morale(사기), penetration(관통), resist(저항)
- 명중/회피: 90% + (민첩 차이) × 2%
- 크리티컬: 순발력/200 확률, (150 + 사기×0.5 + HP↓보너스)% 데미지
- 2회 공격: 속도 1.5배→30%, 2배→60%
- 관통: 방어력 × (1-관통%) 무시
- 저항: 상태이상 저항
- 시나리오/아레나 양쪽에 모두 적용
- 시각 피드백: MISS!/크리티컬!/2회 공격!

### 4. UI 개선
- 아레나 배치: 장수 선택 → 원하는 칸 클릭 (자동 배치 → 수동 배치)
- 3x3 가로 배치 (후열←→전열 마주보는 구조)
- 자동 전투 AUTO 버튼: scrollFactor(0) + create() 마지막에 생성으로 가시성 해결
- 자동 전투 턴 연속: 적군 턴 후 아군 턴 자동 재개
- 자동 전투 액션 메뉴 스킵: 이동 후 바로 행동

### 5. 버그 수정
- AUTO/배속 버튼 안 보이는 문제 (UI 카메라 → scrollFactor + 카메라 ignore 루프 후 생성)
- 자동 전투 1턴만 진행: startEnemyTurn 끝에 auto 재개 추가
- 자동 전투 액션 메뉴에서 멈춤: onMoveComplete에서 autoMode 시 메뉴 스킵
- 가챠 장수 불필요한 기본 스킬: 모든 defaultEquippedSkills를 빈 배열로
