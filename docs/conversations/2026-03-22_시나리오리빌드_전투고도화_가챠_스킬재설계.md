# 시나리오 리빌드 + 전투 고도화 + 가챠 + 스킬 재설계

## 날짜: 2026-03-22 ~ 2026-03-24

## 주요 작업 내역

### 1. 상세 기획 문서 (GDD) 작성
- 원작 조조전 시스템 + 여포 에피소드 조사 (Web 리서치)
- "방구석 여포뎐" GDD 작성: docs/GDD_방구석여포뎐.md
  - 10개 챕터 시나리오 (프롤로그~엔딩 3종)
  - 전투 시스템, 성장, 분기, 수익화
  - 현대인 여포 환생 컨셉 + 4차원 벽 깨기 대사
- 모드별 전투 방식 추가 (PvP 자동배치, 웨이브 던전, 로그라이크 탑)
- 문서 구조 정리: CLAUDE.md(원칙) / GDD(기획) / IMPLEMENTATION.md(구현)

### 2. 시나리오 리빌드
- 제목: "삼국지 조조전" → "방구석 여포뎐"
- 아군 장수: 조조/하후돈/하후연 → 여포/장료/고순
- 프롤로그: 산적 토벌 (튜토리얼, 현대인→여포 환생)
- 챕터 1: 낙양의 야망 (동탁군 선봉 격퇴, 낙양 쟁탈전)
- 챕터 2: 봉의정의 달빛 (호뢰관 전투, 장안 공방전)
- 장료: 보병→기병으로 변경

### 3. 전투 시스템 고도화
- **상성 시스템**: 기병→보병→궁병→기병, 무도가→책사 (1.2x/0.8x)
- **협공**: 인접 아군 있으면 데미지 20% 추가
- **ZOC**: 적 유닛 인접 칸 진입 시 이동 정지
- **보병 8방향 공격**: diagonalAttack 플래그, 체비셰프 거리
- **공격 이펙트**: 근접(검 궤적), 궁병(화살 투사체), 책사(마법진)
- **스프라이트 방향 전환**: 이동/공격 방향에 따라 좌우 반전
- **적군 카메라 추적**: AI 이동/공격/스킬 시 카메라 팬

### 4. UI 개선
- **로비 화면 (LobbyScene)**: 시나리오/자유전투/장수뽑기/장수관리/랭킹/로그아웃
- **장수 관리**: 장수 목록 + 상세 (스탯/스킬/장비)
- **전투 유닛 정보 패널**: 유닛 탭 시 하단 패널 (스탯/스킬 표시)
- **장수 이름 라벨 제거** (전투 맵에서)
- **음소거 버튼**: 모든 씬에 추가, 위치 조정
- **씬 흐름**: TitleScene → LobbyScene → 각 모드

### 5. 가챠 시스템
- **등급 체계**: B/A/S/SR/SSR → N/R/SR/SSR/UR
- **장수 풀**: UR 6명, SSR 8명, SR 8명 = 총 22명
- **확률**: UR 5%, SSR 20%, SR 75% (프리미엄), 천장 90회
- **고유 스킬 14종**: 관우/조조/제갈량/주유/장비/조운 등
- **중복 → 조각 → 금화 변환**
- **가챠 UI**: 1회/10연차, 천장 카운트, 결과 화면

### 6. 스킬 시스템 재설계 (4슬롯)
| 슬롯 | 종류 | 해금 | 진화 기준 |
|------|------|------|-----------|
| 1 | 병종 기본 | 처음 | 장수 레벨 (승급 시 자동) |
| 2 | 장수 고유 | Lv.20 | 각성 단계 |
| 3 | 장착 | 처음 | 스킬 레벨 (스킬서) |
| 4 | 장착 | Lv.10 | 스킬 레벨 (스킬서) |

- 병종 기본 스킬 6병종 x 3단계 = 18종 정의
- 장착 스킬 3단계 진화 (스킬서 + 금화 + 특수재료)
- 장수 고유 스킬: Lv.20 해금, 각성으로 강화

### 7. 승급(전직) 시스템
- Lv.15 1차 승급, Lv.30 2차 승급
- 6병종별 2단계 승급 경로 정의
- 승급 시: 스탯 보너스 + 병종 기본 스킬 자동 진화 + 스킬 해금
