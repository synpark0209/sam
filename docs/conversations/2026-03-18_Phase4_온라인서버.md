# 2026-03-18 | Phase 4 - 온라인 서버

## 구현 완료
- NestJS + TypeORM + PostgreSQL 백엔드
- JWT 인증 (회원가입/로그인, 7일 만료, bcrypt 해싱)
- 서버 세이브/로드 API (캠페인 진행 JSON 저장)
- 랭킹 API (최고 레벨 기준)
- 클라이언트 API 클라이언트 (client/src/api/client.ts)
- 로그인/회원가입 React 모달 (AuthModal)
- CampaignManager에서 localStorage 제거, 서버 전용으로 전환
- TitleScene: 로그인 필수, 서버에서 세이브 확인 후 메뉴 표시

## 서버 구조
```
server/src/
├── main.ts           (CORS, ValidationPipe, 포트 3001)
├── app.module.ts     (TypeORM + ConfigModule)
├── auth/             (register, login, JWT strategy)
├── user/             (User entity, UserService)
└── save/             (GameSave entity, upsert, ranking)
```
