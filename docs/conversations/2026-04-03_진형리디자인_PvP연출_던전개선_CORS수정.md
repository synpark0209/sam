## 주요 작업

### 1. 진형 시스템 완전 재설계
- 진형 패턴을 좌우 방향 기준으로 변경 (col0=후열, col2=전열)
- 모든 진형 5칸 통일 (4칸→5칸), 4명 이상 배치 시 출전 가능
- 옆으로 누운 T자 형태 (--|  |--)
- PvP: 좌우 전투 레이아웃, 던전: 상하 전투 레이아웃 (진형 전치)

### 2. 공유 배치 UI (DeploymentUI.ts)
- PvP/던전 공용 배치 컴포넌트 생성
- 장수 선택: 텍스트 리스트 → PixelLab 스프라이트 4열 그리드
- 탭→자동 배치 (선택→배치 2단계 제거)
- 페이지네이션 (8명/페이지)
- 마지막 진형/장수 모드별 저장 (CampaignProgress.lastFormation)
- 열 라벨 상단 표시 (후열/중열/전열)

### 3. PvP 아레나 전투 연출 강화

- 결투 클로즈업: 스킬 사용 시 공격/피격/사망 PixelLab 애니메이션
- 원거리 유닛 제자리 공격
- 행동 순서 바, 턴 팝업
- 승리/패배 파티클 이펙트
- 클로즈업 후 중복 애니메이션 제거
- PvP 상태 서버 저장 (ELO/전적/일일 티켓)
